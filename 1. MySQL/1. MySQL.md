----
### MySQL
----
1. 오픈 소스의 관계형 데이터 베이스 관리 시스템
2. 다중 스레드, 다중 사용자 형식의 구조 질의어 형식의 데이터 베이스 관리 시스템
3. 공개 소프트웨어이며, 누구나 무료로 다운로드 받아 사용 가능
